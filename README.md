# 🧮 Калькулятор закупки — Птица и мясо

Автономный калькулятор рентабельности закупок. Работает без интернета — один HTML-файл, открывается в любом браузере.

---

## 📁 Структура репозитория

```
📁 purchase-calculator
   📄 calculator.html              ← сам калькулятор (открывать этот файл)
   📄 README.md                    ← описание проекта (этот файл)
   📁 products/
      📄 calculator_products.json  ← база сохранённых товаров (экспорт из калькулятора)
   📁 analytics/
      📄 Аналитика_2026-02.xlsx    ← исходная аналитика по месяцам
      📄 Аналитика_2026-03.xlsx
```

---

## 🚀 Как запустить

1. Скачай файл `calculator.html`
2. Двойной клик — откроется в браузере
3. Интернет не нужен

**Создать ярлык как приложение (Chrome):**
Открой файл в Chrome → три точки → Сохранить и поделиться → Создать ярлык → ✅ Открывать в отдельном окне

---

## 💡 Что умеет калькулятор

- Считает **полную себестоимость на 1 кг** с учётом всех расходов
- Показывает **операционную прибыль и маржу** в реальном времени
- Считает **точку безубыточности** — минимальную цену продажи и максимальную цену закупки
- **Пресеты категорий** — ставки логистики по категориям товаров из реальной аналитики
- **Сохранение товаров** — база товаров со своими ставками
- **Экспорт / импорт** базы товаров в .json файл

---

## 📊 Формула расчёта

```
Операционная прибыль = Выручка
  − Закупка
  − Склад / входящая логистика       (руб/кг, по категории)
  − Доставка клиенту                 (руб/кг, по категории)
  − ФОТ + аренда + ИТ + прочее       (руб/кг, общее по компании)
  − Кредит + банковские услуги        (руб/кг, общее по компании)
  − Дебиторская задолженность         (% в месяц × дни / 30)
```

## 💾 База товаров

Товары сохраняются в браузере (localStorage). Чтобы не потерять:

1. В калькуляторе нажми **"↓ Экспорт в файл .json"**
2. Сохранённый файл положи в папку `products/` этого репозитория
3. При необходимости — **"↑ Импорт из файла .json"** восстановит всю базу

---

## 🔄 Обновление ставок

Ставки офисных расходов и логистики пересчитываются ежемесячно на основе аналитики из 1С.

**Для обновления:**
1. Загрузи новый файл аналитики в папку `analytics/`
2. Передай Claude — он пересчитает ставки и обновит `calculator.html`
3. Загрузи новый `calculator.html` в репозиторий поверх старого
